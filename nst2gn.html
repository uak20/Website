RNA structure in health and disease: investigation of snoRNA clusters in the imprinted Dlk1 Dio3 and Snrpn regions and their role in splicing

Supervisor: Dr. Russell Hamilton, Ferguson-Smith/Hamilton Lab, Department of Genetics, University of Cambridge, CB2 3EH

Abstract

The epigenetic phenomenon of genomic imprinting leads to specific genes being expressed in a parent-specific manner. Imprinted genes (IGs) have key roles in placental and embryonic development as well as an emerging role in brain development and function. The imprinted Dlk1-Dio3 region contains two maternally expressed small nucleolar RNA (snoRNA) clusters, SNORD113 and SNORD114, containing nine and thirty-one copies respectively (Rocha et al., 2008). Human individuals with disruptions in this conserved region display growth retardation and facial dysmorphism (Kagami et al., 2008).

Deletion of the related SNORD116 cluster (paternally expressed) in the Snrpn imprinted region has been shown to be a direct cause of Prader-Willi syndrome (PWS) and postnatal growth retardation, (PNGR), (Sahoo et al., 2008; Skryabin et al., 2007). However, little is known about the mechanism of action of these snoRNAs. Using open-source computational tools, I will model the snoRNA 2D and 3D structures of these regions (SimRNA – Boniecki et al., 2016; ViennaRNA Web Services – Gruber et al., 2008; Vienna RNA – Lorenz et al., 2011). Transcribed as a precursor long non-coding RNA, Rian/MEG8, the C/D box motif-containing snoRNAs of Dlk1-Dio3 have been shown to be directed by genomic imprinting and show brain-specific expression, unusually for snoRNAs (Cavaillé et al., 2000; Rogelj, 2006).

Project Methodology

  1. Retrieved sequences in .fasta format from RNAcentral for the spliceosomal snRNAs of the major spliceosome components: U1, U2, U4, U5 and U6

  2. Spliceosomal snRNAs - Local Pairwise Sequence Alignment

    • Performed on the spliceosomal snRNA sequences previously obtained in 1.
    • Performed using the Smith-Waterman algorithm, via the EBI site.
    • Results
    • This provided identity values for each pair of snRNAs analysed - will be used to construct an identity matrix
    • Identity values tabulated in .csv format
    • These .csv files were uploaded to the CTR-RStudio and read using RStudio
    • Plotted identity values from the matrices to obtain heatmaps using plot.dmat
    • Replotted identity values using ComplexHeatmaps to obtain both clustered and non clustered heatmaps using another R script

  3. Spliceosomal snRNAs - 2D Structural Work

    • .fasta sequence files for the spliceosomal snRNAs were run on the RNAfold server, part of the ViennaRNA Web Server
    • Results
    • Obtained the MFE (minimum free energy) structures in Vienna (dot/bracket) format.
    • Additionally, the centroid structures were obtained (also in Vienna format).
    • foRNA was used on the MFE structures to obtain high quality structural figures.
    • Structural similarity modelling for the various combinations of the spliceosomal snRNAs (as before) was carried out using the RNA distance web server.
    • The results were used to construct a structural similarity matrix
    • As before, these were read and a clustered heatmap was plotted using ComplexHeatmap. I adjusted the code so that the heatmap would be automatically exported as a png image file, incorporating some other aesthetic adjustments
    • FoldAlign was used in order to obtain common 2D elements between the spliceosomal snRNA sequences.
    • Results are in the format of .col files

  4. Spliceosomal snRNAs x snoRNAs

    • An R script was written using PairwiseAlignment (part of Biostrings) in order to carry out local pairwise alignment using fasta files.
    • I tested this script using the Spliceosomal snRNA Sequences.fasta as this allowed the results to be compared to the EBI Outputs to verify. This will be especially useful in the project to speed up comparison of snoRNAs to the spliceosomal snRNAs, as there are a large number of combinations. The script was successful in carrying out the sequence alignments.
    • snoRNA sequences were obtained, and converted to RNA format (for comparison to the spliceosomal snRNAs) using DNA2RNA (part of the FastaUtils R package). 
      DNA2RNA(file=snoRNAs/snoRNA Seqs/SNORD113_hs.fasta, out=snoRNAs/snoRNA Seqs/SNORD113_hs.rna.fasta)
    • Pairwise sequence alignments were set up between the individual spliceosomal snRNAs and the SNORD clusters of the respective species using a new script.
    • These pairwise sequence alignments were run for both human and mouse, and identity matrices were obtained.
    • Using these identity matrices, overall sequence similarity matrices were obtained, grouped by the relevant imprinted region (human Dlk1-Dio3, human Snprn and mouse Snrpn), to obtain new similarity matrices, and heatmaps were thus plotted
    • Another R script written to use RNADistance on the spliceosomal snRNA sequences to calculate 2D structural distances.
    • New matrices, and heatmaps (using ComplexHeatmap) were obtained.
    • Now using the command line to run RNAfold, and RNAdistance (code), I obtained distance matrices for the 2D structures using the snoRNAs vs spliceosomal snRNA pairwise combinations set up earlier. 
      RNAfold -T37 < sno\ x\ Spliceosomal/Pairwise\ Combinations/hU4\ x\ hSNORD113.fasta | RNAdistance -Xm
    • I then collated all the distance values into distance matrices, grouped with respect to the relevant imprinted region (human Dlk1-Dio3, Snrpn and mouse Snrpn), plotting heatmaps.

  5. Searching for Common Elements between snoRNAs and Spliceosomal snRNAs

    • Using Foldalign, which I had previously used as a web server, I proceeded to search for common structural elements between indvidual snoRNAs and the U4 spliceosomal snRNA, as this has been shown to be the key regulator of the major spliceosome.
    • First, the snoRNA sequences were separated into individual snoRNA sequences
    • Next, Foldalign was run to compare the U4 spliceosomal snRNA to the individual snoRNAs of the clusters using this code, which outputs a list of hits for each combination: 
      cd ~/U4\ x\ snoRNAs/Sequences/

foldalign -plot_score hU4.fasta hSNORD113_1.fasta | locateHits > ~/U4\ x\ snoRNAs/Hit\ List\ hU4_hSNORD113/hU4_hSNORD113_01.txt
    • For each SNORD cluster's hit lists, a "filtered hit list" was created, of all the hits with a p score of less than 1.000 - a collated form of these lists (human and mouse) will then be used to calculate the mean p score for the top 5 hits of an individual snoRNA - separate lists (human and mouse) were made of the mean scores of the top 5 hits.
    • The 10 snoRNAs with the lowest means are the candidate snoRNAs that will be further analysed in 3D modelling, as these are those snoRNAs with most similarity to the U4 spliceosomal snRNA.

  6. Duplex Formation Prediction

    • RNAcofold was used to predict duplexes, as well as their ∆G of formation (binding)
    • Predicted duplexes' structures
    • Graphs representing ∆G of duplex formation for candidate snoRNAs with U4, U5, and U6 spliceosomal snRNAs.

  7. 3D Structure Prediction

    • Candidate snoRNA secondary structures predicted using RNAfold
    • Spliceosomal snRNAs' secondary structures predicted using RNAfold
    • Restraints may be assigned to any pair of atoms, however through specifying 2D structure (in dot/bracket, Vienna notation), base pairing restraints are created (using this script between pairs of atoms that form the link between the triangular pseudo-bases. This makes use of a Perl script kindly provided by my supervisor Dr R.S. Hamilton
    • Run SimRNA, a coarse grained 3D structural prediction program, for the Candidate snoRNAs and the Spliceosomal snRNAs
    • Visualise the 3D structures in PyMol.
    • Align candidate snoRNA structures to their respective U4 spliceosomal snRNA using this script in PyMol.
